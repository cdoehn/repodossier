# Planning

This directory contains current planning notes and archived implementation history.

RepoDossier 1.0.0 is complete. Its original roadmap and milestone implementation notes are preserved in:

    planning/archive/1.0.0/

The versioned archive layout is intentional. Future release-planning archives can be added next to it, for example:

    planning/archive/1.1.0/
    planning/archive/2.0.0/

Archive naming convention:

    planning/archive/<version>/roadmap.md
    planning/archive/<version>/milestone2.md
    planning/archive/<version>/milestone3.md
    planning/archive/<version>/milestone4.md

Use lowercase milestone filenames inside archives. Do not use names such as `MILESTONE4.md` in versioned archives.

Files in `planning/archive/` are historical release records. They are useful for understanding how a release was built, but they are no longer the active roadmap or the source of current release status.

Current forward-looking work lives in:

    planning/roadmap_next.md

User-facing usage documentation lives in the repository root README.md.

Shared project specification:

    planning/spec.md
