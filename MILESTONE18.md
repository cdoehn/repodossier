# Milestone 18 – Release 1.0

Milestone 18 finalizes RepoContext as release version 1.0.0.

## Scope

This milestone is a release-hardening milestone. It does not introduce a new major analysis feature. It completes the release by validating documentation, smoke tests, pipx installation, CLI version output, and final export behavior.

## Completed release areas

### 18.1 Release scope and project state

- The public CLI surface is validated through help checks.
- Project metadata is prepared for release.
- The release version is set to 1.0.0.
- Deprecated project snapshot workflows are not part of the release workflow.

### 18.2 Documentation completion

- README release usage documentation is included.
- Installation instructions are documented.
- pipx usage is documented.
- Development installation is documented.
- Main commands are documented.
- Export files are documented.
- Configuration usage is documented.
- Secret masking limitations are documented.
- Bash support is documented.
- Static-analysis limitations are documented.

### 18.3 Smoke tests

Release smoke tests are available in:

    tests/test_release_smoke_cli.py

They validate the main CLI export flows:

    repocontext full
    repocontext export-ai
    repocontext export-docs
    repocontext changed

The smoke tests create a temporary Git repository and verify that the expected export files are produced.

### 18.4 pipx validation

Isolated pipx validation is available in:

    scripts/validate_pipx_release.sh

The validation script installs RepoContext into a temporary pipx home, checks CLI help output, creates a temporary Git repository, runs the release export commands, verifies output files, and checks reinstall behavior.

### 18.5 Version 1.0.0

RepoContext now reports:

    repocontext 1.0.0

via:

    repocontext --version

Version tests are available in:

    tests/test_version_cli.py

### 18.6 Final release checks

Final release validation should include:

    python3 -m pytest --color=yes
    python3 -m pytest --color=yes tests/test_release_smoke_cli.py tests/test_version_cli.py
    scripts/validate_pipx_release.sh
    repocontext --version
    repocontext --help
    repocontext full --help
    repocontext export-ai --help
    repocontext export-docs --help
    repocontext changed --help

Final export validation should be done with RepoContext itself:

    repocontext full
    repocontext export-ai
    repocontext export-docs
    repocontext changed

Generated export artifacts such as full.txt, ai.txt, docs.txt, and changed.txt are local outputs and should normally stay uncommitted.

## Release result

Milestone 18 is complete when:

- the full pytest suite is green
- release smoke tests are green
- pipx validation is green or documented as locally unavailable because pipx is missing
- repocontext --version returns repocontext 1.0.0
- full.txt, ai.txt, docs.txt, and changed.txt can be generated by RepoContext itself
- git status contains only intentional tracked release changes before committing
- the final release commit is created
